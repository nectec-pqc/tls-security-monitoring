# Gathering Monitoring Targets

**Objective:** Compile a database of public websites and endpoints of interest
so we can monitor their TLS security settings using our main project. Larger
number of targets should give us more reliable and useful statistics.

## Tools

The following external tools can be used to list subdomains from open-access sources

- [assetfinder](assetfinder)
- [subfinder](subfinder)

See subdirectory for more information on how to use them.
